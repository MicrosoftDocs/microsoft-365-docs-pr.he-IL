# <a name="transition-a-microsoft-365-business-csp-subscription"></a>מעבר מנוי Microsoft CSP עסקית של 365

אם יש לך מנוי CSP תצוגה מקדימה של העסק 365 של Microsoft, בצע את המדריך כדי לברר כיצד תוכל להעביר את המנוי הקיים שלך בתצוגה מקדימה כדי גה עסקיים 365 של Microsoft (זמינות כללי).

**אופן המעבר מנוי תצוגה מקדימה לגה**

1. היכנס למרכז <a href="https://partnercenter.microsoft.com" target="_blank">שותף</a>.
2. מלוח המחוונים, בחר **לקוחות**, ולאחר מכן חפש ובחר את שם החברה.

    המנויים עבור החברה יפורטו.

    ![המנויים של הלקוח במרכז שותף](images/pc_customer_subscriptions_1.png)
    
3. בדף **מנויים** של החברה, בחר **הוסף מנוי**.
4. **מנוי חדש** בעמוד, בחר **לעסקים קטנים** ולאחר מכן בחר **עסקיים 365 Microsoft** מתוך הרשימה.
5. הוסף את מספר רשיונות ולאחר מכן בחר **הבא: סקירת** כדי לסקור את המנוי ולאחר מכן בחר באפשרות **שלח**.

    ![סקור את מנוי חדש לעסק 365 של Microsoft](images/pc_customer_reviewnewsubscription.png)

    **מנויים מבוסס רשיון** יציג **Microsoft 365 עסקיים**ותצוגה **מקדימה עסקית של 365 Microsoft** . יהיה עליך להשעות את המנוי תצוגה מקדימה הבא.

6. בחר **תצוגה מקדימה של העסק 365 Microsoft**.
7. **תצוגה מקדימה של Microsoft 365 ביקור** בדף, בחר **מושעה** להשעות את המנוי תצוגה מקדימה.

    ![להשעות את מנוי Microsoft תצוגה מקדימה של העסק 365](images/pc_customer_m365bpreview_suspend.png)

8. בחר באפשרות **שלח** כדי לאשר.

    בעמוד **מנויים** , אשר מצב **התצוגה המקדימה של Microsoft 365 עסקיים** מציג **מושעה**.

    ![לאשר שמצב מנוי תצוגה מקדימה מושעה](images/pc_customer_m365bpreview_suspend_confirm.png)

9. באופן אופציונלי, באפשרותך גם לאמת את הסכם הרשיון. לשם כך, בצע את הפעולות הבאות:
    1. בחר **רשיונות ומשתמשים** מדף **מנויים** של החברה.
    2. מדף **רשיונות ומשתמשים** , בחר משתמש.
    3. בדף של המשתמש, עיין בסעיף **להקצות רשיונות** ואשר הוא מציג **Microsoft 365 עסקיים**.

        ![לאשר שרשיון העסק 365 Microsoft מוקצית למשתמש](images/pc_customer_userslicenses_m365b_validate.png)
